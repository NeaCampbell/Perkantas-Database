##Tutorial to Enter VPS
1. download and run putty.
2. run putty, and go to Session group.
3. type the IP address for the VPS.
4. choose SSH as the connection type.
5. go to Connection group, select SSH section, and enter Auth section.
6. at Auth section, browse private_key.ppk.
7. back to Session group, and Save the session using any name suitable for you.
8. click open, and enter root as user and Tamanwilis as the passphrase.
9. Enjoy!
